# Wizard Icon Component

The wizard icon is a wrapper of the [Rounded Icon](?path=/docs/components-assets-rounded-icon--story-010-icon) that encapsulates the styles to expose an API with the states
of the icon in the context of the wizard.

<br />

## Properties

### state

Currently, the icon could have 3 different states: `completed`, `pending` and `disabled`

### step

From 1 to 6, the step number of the icon
