# Paragraph Component

This component can be used to render running text. Its a wrapper around the `<p>` tag.

### Example:

<Canvas withSource="open" withToolbar={false}>
  <Story id="components-content-paragraph--story-010-default" />
</Canvas>

NOTE: All attributes are optional. The component can simply be used like this without specifying any attributes:

<Source code={`<b2b-paragraph>Simple Paragraph</b2b-paragraph>`} />
